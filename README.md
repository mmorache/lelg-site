# Loose Ends, Long Goodbyes – Website

This repository contains the source for the **Loose Ends, Long Goodbyes** website.

The site is a lightweight static site built with **Jekyll** and hosted via **GitHub Pages**.

## Domain
https://looseendslonggoodbyes.com

## Tech Stack
- Jekyll (GitHub Pages compatible)
- Markdown content
- Static assets
- No backend / database

## Purpose
This site serves as the online home for the Loose Ends, Long Goodbyes music project:
- music / demos
- updates
- contact information
- general presence

## Development
Local development is optional. Most changes can be made directly in Markdown and pushed.
